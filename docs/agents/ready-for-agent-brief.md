## AFK/HITL decision guide

Use `ready-for-agent` only if the agent can finish and verify without extra human input. Use `ready-for-human` if the task requires:

- live Sales Navigator or LinkedIn mutation
- browser login by the Operator
- judgment about acceptable lead quality trade-offs
- external account/customer data not already sanitized into fixtures
- budget, credential, or account changes
- publishing/sending with real external consequences
